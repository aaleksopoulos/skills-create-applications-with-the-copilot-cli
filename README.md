## CLI Calculator

A small Node.js CLI calculator implementing basic arithmetic operations: addition, subtraction, multiplication, and division.

Usage examples:
- node src/calculator.js add 2 3        # -> 5
- node src/calculator.js subtract 5 2   # -> 3
- node src/calculator.js multiply 4 3   # -> 12
- node src/calculator.js divide 8 2     # -> 4

The module exports the functions: addition, subtraction, multiplication, division (see src/calculator.js).
Division by zero prints an error message and returns a non-zero exit code.
